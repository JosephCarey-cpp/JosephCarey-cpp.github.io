Frontend

About:

The Frontend of the Sustainable Garden Project App is to give the user a friendly interface to view data from the sustainable garden. The data shown to the user can be used for other departments in Gavilan College, such as the Biology Department.

Using Nativescript and Angular, the app works on android devices, and is constantly being updated with new features as the project progresses.

Check out the frontend on GitHub in the GitHub Project Tab!

Backend

About:

The Backend of the Sustainable Garden Project App is a combination of using Docker containers to store a Django database, which can be utilized on any platform. Docker is an essential aspect of this project, as the server will run on a Raspberry Pi 4B, which can have its OS change. Since the server is within Docker containers, it does not matter where deployed the server on any system.

The database can showcase information about the sensors coming from the garden through an Arduino MEGA board and send the data to the app with serialization. The server also has user account storage for users to get specific or custom data changes.

Check out the frontend on GitHub in the GitHub Project Tab!
