Gavilan College & San Jose State University (SJSU) Summer Internship

Description:

DURATION: JULY 2021 - AUG 2021; aprox. 6 - 8 weeks

This project was the first internship I participated in my career. The project's goal was to find a solution to breaks in 3D-printed parts using a continuous stream of filament with an algorithm. The project called for using the research of existing algorithms and creating a new one that fits the solution to the project.

To see a presenation about the project, check out the Video Tab

During this time, the project has helped in gaining the following skills:

  • Algorithm creation using research and problem solving skills
  • Knowledge of graphs; specifically Eulerian and Planar graphs
  • Project Cycle
